# Daniel's Neovim Config

LazyVim-based Neovim configuration with custom Tachikoma dashboard.

## Requirements

- [Neovim](https://neovim.io/) >= 0.9.0
- [chafa](https://hpjansson.org/chafa/) - Terminal graphics for dashboard image

### Installing chafa

**macOS:**
```bash
brew install chafa
```

**Ubuntu/Debian:**
```bash
sudo apt install chafa
```

**Arch Linux:**
```bash
sudo pacman -S chafa
```

**Fedora:**
```bash
sudo dnf install chafa
```

## Features

- Tokyo Night theme with custom dark background
- Tachikoma (Ghost in the Shell) dashboard image
- Noice.nvim disabled for cleaner UI
